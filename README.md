# Smart Climate Controller (UA)

Розумний контролер клімату для Home Assistant, спеціально розроблений для ефективного керування кондиціонерами та мультиспліт-системами.

Ця інтеграція перетворює ваші звичайні кондиціонери на інтелектуальну кліматичну систему, яка не просто вмикає/вимикає пристрої, а динамічно керує їхньою температурою для швидкого досягнення комфорту та економії енергії.

## 🌟 Основні можливості

- **Розумне керування мультиспліт-системами**: Автоматична синхронізація режимів (Heat/Cool) для всіх внутрішніх блоків однієї групи.
- **Динамічна корекція температури**: Система автоматично змінює цільову температуру на самому кондиціонері (наприклад, виставляє +32°C при потребі швидкого нагріву), щоб він працював на повну потужність до досягнення цілі.
- **Інтелектуальний AUTO режим**: Автоматичне перемикання між обігрівом та охолодженням на основі зовнішньої температури та реальних потреб у кожній кімнаті.
- **Гнучкі пресети**:
  - 🌿 **Eco**: Максимальна енергоефективність з розширеними порогами та раннім вимкненням.
  - 🏠 **Comfort**: Точне підтримання заданої температури.
  - 🚀 **Boost (Heat/Cool)**: Максимально швидкий нагрів або охолодження за 5 хвилин.
- **Захист обладнання**: Вбудовані таймери для запобігання занадто частому вмиканню компресора або зміні режимів.
- **Повна підтримка UI**: Налаштування всіх параметрів через зручний інтерфейс Home Assistant.

## 🛋️ Пресети та режими роботи

### 1. Comfort (Комфорт)
Основний режим для щоденного використання. Система підтримує температуру з високою точністю (стандартний гістерезис ±0.5°C), використовуючи лінійну або ступінчасту корекцію.

### 2. Eco (Еко) 🌿
Режим для максимальної економії:
- **Розширені пороги**: Кондиціонер вмикається рідше завдяки збільшеному вдвічі гістерезису.
- **Раннє вимкнення**: Система вимикає AC відразу при досягненні цільової температури, не чекаючи інерційного перегріву.
- **Знижений пріоритет**: Кімнати в Eco режимі мають меншу вагу при прийнятті рішення про зміну режиму всієї мультиспліт-системи.
- **Еко-зміщення**: Можливість задати додатковий температурний офсет для роботи AC у "м'якому" режимі.

### 3. Boost (Форсований режим) 🚀
Потрібно швидко нагріти або охолодити кімнату?
- Активується на 5 хвилин (налаштовується).
- Виставляє екстремальні значення температури на AC для максимальної продуктивності.
- Після завершення автоматично повертає попередній пресет (Eco або Comfort).

## ⚙️ Налаштування

Параметри розділені на логічні блоки для зручності:

### 🌡️ Датчики та Пороги
- **Зовнішня температура**: Використовується для блокування невідповідних режимів (наприклад, заборона обігріву в спеку).
- **Поріг тільки ОБІГРІВУ**: Якщо на вулиці нижче цього значення, система не дозволить увімкнути охолодження.
- **Поріг тільки ОХОЛОДЖЕННЯ**: Якщо на вулиці вище цього значення, обігрів буде заблокований.

### 📉 Корекція (Алгоритм)
- **Minor Correction (Дрібна)**: Невелике зміщення температури на AC (наприклад, +5°C до цілі), коли до мети залишилося небагато.
- **Major Correction (Сильна)**: Максимальне зміщення (наприклад, +10°C), коли в кімнаті занадто холодно/спекотно.
- **Linear Correction**: Плавна зміна температури на AC для максимально непомітної підтримки клімату.

### 🛡️ Таймери захисту
- **Min Mode Switch**: Мінімальний час між перемиканням всієї системи з HEAT на COOL (захист компресора).
- **Min Power Switch**: Мінімальний час між увімкненням та вимкненням живлення AC.

## 🚀 Встановлення

1. Встановіть через **HACS** або скопіюйте вміст `custom_components/smart_climate_controller` у папку вашого Home Assistant.
2. Перезапустіть Home Assistant.
3. Перейдіть у **Налаштування -> Пристрої та служби -> Додати інтеграцію**.
4. Знайдіть **Smart Climate Controller**.
5. Дотримуйтесь інструкцій на екрані для додавання вашої першої кімнати.

## 📖 Як це працює (Технічні деталі)

Система працює циклічно (кожні 30 секунд). Координатор збирає дані з усіх датчиків, обчислює "вагу" (пріоритет) потреб кожної кімнати та приймає рішення про роботу всієї групи.
